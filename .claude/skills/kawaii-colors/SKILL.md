# KAWAII デザインスタイル

## 完成イメージ（最重要！）

- **おもちゃ箱のような「無害で楽しい」印象**
- **頭身の低いコロコロした丸っこいキャラクター**
- **パステルネオンな色使い**
- **ポップでドリーミーな世界観**

## キャラクターデザイン原則

どんなゲームでも必ず守る：
- **丸みのあるフォルム** - 角張った形は避ける
- **大きな目、ほっぺ、表情** - 愛嬌を出す
- **頭が大きく体が小さい** - デフォルメ体型
- **明るい色** - 黒・暗い色は使わない

## カラーパレット

```
背景:   #FFF0F5 (淡いピンク)
プレイヤー: #FF69B4 (Hot Pink)
敵:     #9370DB (Pastel Purple)
アイテム: #4ECDC4 (Mint Green)
スコア:  #FFD700 (Gold)
```

## 禁止

- 黒色、暗い色、リアルな質感
- 無機質・かわいくないデザイン
- 単純な四角形・三角形だけのキャラ
