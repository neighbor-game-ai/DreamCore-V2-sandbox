# Resume - 作業再開スキル

前回のセッションからの続きを確認し、次の作業を提案します。

## 実行手順

1. 以下のファイルを読んで現状を把握:
   - `TODO.md` - 未完了タスク一覧
   - `CLAUDE.md` - プロジェクト規約とPhase 1完了ステータス

2. 最近のコミットを確認:
   ```bash
   git log --oneline -5
   ```

3. 現状をサマリーで報告:
   - 完了済み項目
   - 未完了の優先タスク
   - 技術的負債

4. ユーザーに次の作業を確認:
   - TODO.mdの優先度高タスクを提案
   - または別の作業があれば聞く

## 出力フォーマット

```
## 現状サマリー

**完了済み**: ...
**未完了（優先度高）**: ...
**最近のコミット**: ...

## 次の作業候補

1. [優先度高] ...
2. [優先度高] ...
3. [Phase 2] ...

どれから始めますか？または別の作業がありますか？
```
