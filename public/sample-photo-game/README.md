# Photo Game Creator Sample

作成日: 2026-02-05
目的: 初心者向けゲーム作成フローのプロトタイプ。写真をアップロードしてパーソナライズされたゲームを作成する体験をテスト。

## 概要

DreamCore 本体に統合する前の独立したサンプル実装。
ユーザーが自分の写真を使ってゲームを作る「初心者向けフロー」を検証する。

## フロー

1. 写真をアップロード
2. 「人物全体」or「顔だけ」を選択
3. ゲームタイプを選択（シューティング/アクション/パズル/その他）
4. プロンプト生成 → ゲーム作成

## 内容

| ファイル | 説明 |
|---------|------|
| `index.html` | メインUI（4ステップのウィザード形式） |
| `style.css` | スタイル（モバイル対応、グラデーション背景） |
| `app.js` | ロジック（顔検出、画像処理、プロンプト生成） |

## 技術

- **顔検出**: face-api.js（TinyFaceDetector）
- **背景除去**: モック実装（将来 BRIA RMBG 連携予定）
- **UI**: Vanilla JS、CSS Grid/Flexbox

## 確認方法

```
http://localhost:3000/sample-photo-game/
```
